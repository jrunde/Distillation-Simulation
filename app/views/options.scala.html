  • FINDING THE FUEL THAT FITS

    Distillation Curve Simulation

    Engineers at the Great Lakes Bioenergy Research Center (GLBRC) are working to design mixtures of renewable fuels made from plants (biofuels) that could replace existing fossil fuels like gasoline. Done right, these drop-in biofuels could be used in our current engines without any modifications. Done wrong, poorly designed biofuels can cause engine problems and produce harmful emissions.

    A key characteristic of a fuel is its volatility (ability to vaporize). To measure a fuelâ€™s volatility, engineers slowly heat the fuel in a distillation apparatus and measure what % of the liquid evaporates over time (Fig. 1). Then they plot a graph of temperature vs % of fuel evaporated (Fig. 2). This graph is called a distillation curve. If they can design a biofuel mixture that has a distillation curve that is very similar to the reference fuel, they they know that the biofuel will be a good fit to replace the reference fossil fuel.

    In this simulation, you will take on the role of a fuel engineer designing biofuel mixtures to replace fossil fuels. At each level, you will be given a distillation curve for a reference fossil fuel. Then will have analyze the graph and select a mixture biofuel options that you think will match the volatility of the reference fuel. After the distillation simulation runs, you can see how well your biofuel distillation curve matches the reference fuel. Each curve you generate is scored from 0-100 based on how well it matches the reference fuel. A score of at least 80 is needed to move to the next level.

    The number of chemical components of the reference fuel and the number of biofuel options to mix increase as you go to higher levels:

    Level Description
    1 Select from 5 biofuel component options to match distillation curve for a 1-component reference fuel.
    2 Select from 10 biofuel component options to match distillation curve for a 1-2 component reference fuel.
    3 Select from 15 biofuel component options to match distillation curve for a 2-3 component reference fuel.
    4 Create a biofuel mixture using up to four of the 15 biofuel component options to match the distillation of actual gasoline.
